# 04-导线与互连模型

本章对应教材第 4 章“导线（The Wire）”，用于整理片上互连线的寄生电容、电阻、电感，以及导线电路模型。它连接前一章 [[../03-器件与MOS晶体管模型/README|器件与 MOS 晶体管模型]] 和后续 [[../05-CMOS反相器/README|CMOS反相器]]、[[../08-延迟与功耗/README|延迟与功耗]]。

## 本章目标

- 理解为什么深亚微米工艺中互连线不再是“理想导线”。
- 掌握互连线的电容、电阻和电感来源。
- 学会判断什么时候可以忽略某些寄生效应。
- 建立从导线几何尺寸到电路模型的基本映射。
- 为后续 RC 延迟、串扰、传输线和 SPICE 互连模型打基础。
