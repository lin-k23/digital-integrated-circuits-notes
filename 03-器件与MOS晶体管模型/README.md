# 03-器件与 MOS 晶体管模型

本章对应教材第 3 章“器件”，用于建立后续 [[../05-CMOS反相器/README|CMOS反相器]]、[[../06-CMOS组合逻辑/README|CMOS组合逻辑]] 和 [[../08-延迟与功耗/README|延迟与功耗]] 的器件基础。

## 本章目标

- 理解二极管（diode）和 pn 结的静态、动态特性。
- 理解 MOS 晶体管（MOSFET）为什么可以近似为电压控制开关。
- 掌握阈值电压（threshold voltage）、体效应（body effect）、线性区、饱和区、亚阈值区等概念。
- 建立适合手工分析的 MOS 等效模型。
- 理解寄生电容、源漏电阻和工艺偏差对数字电路的影响。
