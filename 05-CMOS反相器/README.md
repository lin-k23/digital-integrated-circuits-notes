# 05-CMOS反相器

本章用于整理 CMOS 反相器（CMOS inverter）的静态特性、动态特性和功耗模型。

## 重点主题

- 电压传输特性（voltage transfer characteristic, VTC）
- 开关阈值
- 噪声容限（noise margin）
- 传播延迟（propagation delay）
- 上升时间和下降时间
- 动态功耗、短路功耗、静态功耗
