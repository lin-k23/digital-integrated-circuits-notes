# 05-CMOS反相器

本章对应教材第 5 章“CMOS 反相器（The CMOS Inverter）”，用于整理静态 CMOS 反相器的逻辑功能、电压传输特性、噪声容限、动态延迟和功耗模型。

## 本章目标

- 理解 CMOS 反相器（CMOS inverter）为什么是数字电路的核心基本单元。
- 掌握静态 CMOS 反相器的开关直觉：NMOS 下拉、PMOS 上拉。
- 掌握电压传输特性（voltage transfer characteristic, VTC）、开关阈值（switching threshold）和噪声容限（noise margin）。
- 理解传播延迟（propagation delay）与负载电容、器件等效电阻、晶体管尺寸之间的关系。
- 掌握反相器尺寸优化、self-loading、effective fanout 和缓冲链设计直觉。
- 建立动态功耗、短路功耗、静态功耗、能量和 energy-delay 的复习框架。
- 理解工艺缩放对反相器速度、能量、泄漏、噪声容限和互连影响的共同作用。
- 为后续 [[../06-CMOS组合逻辑/README|CMOS组合逻辑]] 的逻辑门设计提供模板。
