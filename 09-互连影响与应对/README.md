# 09-互连影响与应对

本章对应教材第 9 章 “Coping with Interconnect”，用于整理互连寄生对性能、可靠性和供电噪声的影响，以及对应的电路和版图应对策略。

## 本章目标

- 从电容、电阻、电感三个角度理解互连寄生如何影响数字集成电路。
- 掌握电容串扰（capacitive crosstalk）的可靠性风险和延迟影响。
- 掌握大电容负载驱动、缓冲链、I/O pad driver、三态驱动等设计直觉。
- 后续继续整理 IR drop、电迁移、RC delay、repeater insertion、wire pipelining、ground bounce、transmission line termination、reduced-swing 和 NoC。
