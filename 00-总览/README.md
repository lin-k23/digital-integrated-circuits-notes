## 整理策略

由于当前 PDF 为英文文字版，整理时采用小步推进：

1. 先建立章节框架。
2. 每次只整理一个小节或一组紧密相关知识点。
3. 公式、变量下标和不等式方向以英文 PDF 为准。
4. 图片不裁剪提交，只保留占位符。
5. 不确定内容用“需核对”标记。
