# 数字集成电路复习笔记

Obsidian-style review notes for Digital Integrated Circuits.

本仓库用于整理《数字集成电路》课程复习笔记，重点服务于：

- 课程复习
- 考前速查
- 概念理解
- 公式整理

## 整理原则

- 中文术语优先，英文术语放在括号中。
- 每章使用独立目录和 `README.md` 作为入口。
- 知识点拆分为独立 Markdown 文件。
- 使用 Obsidian `[[内部链接]]` 建立双向链接。
- 公式使用 LaTeX 块。
- 扫描版 PDF 中识别不确定的公式、变量、图注必须标注“需核对”。
- 必要图片仅保留占位符，后续人工补图。
