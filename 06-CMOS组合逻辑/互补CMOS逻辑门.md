# 互补 CMOS 逻辑门

## 一句话理解

互补 CMOS 逻辑门（complementary CMOS gate）用 NMOS 下拉网络实现输出为 0 的条件，用 PMOS 上拉网络实现输出为 1 的条件；二者是对偶网络，因此稳态下输出总能被强拉到 $0$ 或 $V_{DD}$。

## 从反相器到多输入门

CMOS 反相器是最简单的互补 CMOS 门：

- 输入高时，NMOS 导通，PDN 把输出拉低。
- 输入低时，PMOS 导通，PUN 把输出拉高。

多输入互补 CMOS 门只是把单个 NMOS/PMOS 替换成 NMOS/PMOS 网络：

```text
反相器：一个 PMOS + 一个 NMOS
多输入门：PMOS 网络 + NMOS 网络
```

## 强 0 与强 1

互补 CMOS 使用 NMOS 构造 PDN、PMOS 构造 PUN，核心原因是器件传输强弱不同：

- NMOS 传输强 0（strong zero），适合把节点拉到 GND。
- PMOS 传输强 1（strong one），适合把节点拉到 $V_{DD}$。

若用 NMOS 上拉高电平，输出最多只能到：

$$
V_{DD}-V_{Tn}
$$

若用 PMOS 下拉低电平，输出最多只能降到：

$$
|V_{Tp}|
$$

因此互补 CMOS 的自然选择是：

```text
NMOS → 下拉网络 PDN
PMOS → 上拉网络 PUN
```

## 互补 CMOS 的输出性质

由于 PUN 与 PDN 互补：

- 当输出应为 1 时，PUN 导通，PDN 关断。
- 当输出应为 0 时，PDN 导通，PUN 关断。

因此输出不会靠电容暂存，而是稳态连接到低阻电源轨。

这带来：

- 好噪声容限。
- 满摆幅输出。
- 理想静态无直流功耗。
- 易于级联和自动综合。

## 互补 CMOS 天然实现反相逻辑

一个重要结论：互补 CMOS 单级门天然实现反相型逻辑，例如：

- NAND
- NOR
- XNOR

而非反相型逻辑如 AND、OR、XOR，通常需要额外一级反相器，或者用其他逻辑风格实现。

原因是 PDN 直接实现“输出为 0 的条件”，最终输出是该条件的反函数。

## 晶体管数量

如果逻辑函数有 $N$ 个输入，互补 CMOS 通常需要：

$$
2N\ \text{transistors}
$$

每个输入控制一个 NMOS 和一个 PMOS。

优点是鲁棒，缺点是面积和输入电容较大。

## 设计直觉

- 优先用互补 CMOS 实现普通标准单元。
- 对复杂逻辑，先构造 PDN，再通过对偶得到 PUN。
- 单级复杂门不一定越复杂越好，过多串联管会增加延迟。
- 如果需要非反相函数，常在互补 CMOS 反相型门后加一级反相器。

## 易错点

- 不要用 NMOS 做上拉、PMOS 做下拉来构造满摆幅静态 CMOS 门。
- 不要忘记 PUN 和 PDN 必须对偶，否则可能出现浮空或直通电流。
- 不要把互补 CMOS 的逻辑电平和尺寸比绑定；尺寸影响速度和阈值，不影响理想稳态电平。
- 不要忽略输入电容：每个输入都同时驱动一个 NMOS 和一个 PMOS。
