# 02-制造工艺与版图规则

本章对应教材第 2 章 “The Manufacturing Process”，用于整理 CMOS 制造工艺、光刻与掩膜、常见工艺步骤、设计规则、版图约束、封装和工艺发展趋势。

## 本章目标

- 理解为什么数字电路设计者仍需要了解制造工艺。
- 掌握光刻掩膜（optical masks）在“设计图形 → 硅片结构”之间的桥梁作用。
- 掌握 CMOS 制造中常见步骤：氧化、光刻、扩散/离子注入、淀积、刻蚀、平坦化、金属化和钝化。
- 理解 n-well / dual-well CMOS 工艺中 NMOS、PMOS 的形成方式。
- 理解设计规则（design rules）为什么是设计者与工艺工程师之间的 contract。
- 理解 IC 封装如何影响性能、供电、散热、可靠性和成本。
- 理解 Copper / low-k / SOI / 3D integration 等工艺趋势如何缓解互连、寄生和系统集成瓶颈。
- 建立后续学习版图、寄生参数、互连、封装和工艺趋势的入口。
