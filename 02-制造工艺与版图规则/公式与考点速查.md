# 公式与考点速查

> 本文件用于汇总第 2 章制造工艺与版图规则的关键定义、数量级和高频考点。当前已覆盖 2.1–2.4 的制造流程、光刻、常见工艺步骤、CMOS 简化流程、设计规则、IC 封装和封装热设计。

## 核心概念

### optical masks

光学掩膜是制造工艺与版图设计之间的接口：

```text
layout polygons -> optical masks -> wafer patterns -> devices and wires
```

掩膜定义不同工艺层的图形，最终形成器件和互连。

### design rule set

设计规则规定最小宽度、最小间距和层间约束，是：

```text
circuit designer 和 process engineer 之间的 contract
```

遵守设计规则意味着版图在该工艺下具有可制造性保证。

## 硅片与阱结构

典型硅片数量级：

- 直径：4–12 inch，约 10–30 cm。
- 厚度：不超过约 1 mm。
- 材料：single-crystalline, lightly doped wafer。

CMOS 器件体区：

| 器件 | 所需体区 |
|---|---|
| NMOS | p-type substrate / p-well |
| PMOS | n-type substrate / n-well |

dual-well CMOS：

```text
NMOS in p-well
PMOS in n-well
```

## 光刻循环

典型 photolithography cycle：

```text
oxidation layering
-> photoresist coating
-> stepper exposure
-> photoresist development and bake
-> etching
-> spin/rinse/dry
-> process step
-> photoresist removal / ashing
```

正胶/负胶：

| 类型 | 曝光后性质 |
|---|---|
| negative photoresist | 曝光区域变为不溶 |
| positive photoresist | 曝光区域变为可溶 |

## 常见工艺步骤

### 扩散

典型扩散温度：

$$
900\sim1100^\circ C
$$

特点：掺杂剂同时垂直和横向扩散，浓度从表面向内部下降。

### 离子注入

控制关系：

```text
ion acceleration -> penetration depth
beam current + exposure time -> dosage
```

离子注入优点是 depth 与 dosage 可相对独立控制。

### 退火

离子注入后的晶格损伤可通过退火修复：

$$
T\approx1000^\circ C,\qquad t\approx15\sim30\ \mathrm{min}
$$

### CVD / sputtering / CMP

- CVD：化学气相淀积，用于 $Si_3N_4$、polysilicon 等。
- sputtering：常用于 Aluminum 互连层。
- CMP：chemical-mechanical planarization，用于多层互连前的平坦化。

## CMOS 简化流程

简化 dual-well CMOS flow：

```text
Define active areas / etch and fill trenches
-> implant well regions
-> deposit and pattern polysilicon
-> implant source/drain and substrate contacts
-> create contact/via windows
-> deposit and pattern metal layers
-> passivation and pad openings
```

## self-aligned process

多晶硅 gate 先图形化，再进行 source/drain implant。gate 会遮挡 channel region，使 source/drain 的位置相对 gate 精确确定。

```text
polysilicon gate defines channel position
```

这称为 self-aligned process。

## 设计规则

### lambda rules

可缩放设计规则通常使用单一参数 $\lambda$：

$$
\text{minimum line width}=2\lambda
$$

对 $0.25\ \mu m$ 工艺：

$$
2\lambda=0.25\ \mu m
$$

$$
\lambda=0.125\ \mu m
$$

### micron rules

micron rules 直接使用绝对尺寸：

```text
width / spacing / overlap in μm
```

工业设计通常使用 micron rules，因为它能更充分利用具体工艺能力。

## 版图层

从设计者角度，CMOS 版图层包括：

- wells / substrate。
- diffusion / active areas。
- select layers。
- polysilicon layers。
- metal interconnect layers。
- contact / via layers。

晶体管形成方式：

```text
MOS transistor = diffusion layer ∩ polysilicon layer
```

金属层互连：

```text
metal_i ∩ via ∩ metal_{i+1}
```

## contact / via 数量级

教材 0.25 μm 工艺示例：

- contact hole 最小尺寸约 $0.3\ \mu m$。
- poly / diffusion 需要延伸约 $0.14\ \mu m$ 覆盖 contact hole。
- 最小 contact 区域约：

$$
0.44\ \mu m\times0.44\ \mu m
$$

> 需核对：contact/via 具体数值依赖工艺和版图规则，考试或作业中应以题目/PDK 为准。

## IC 封装

### Rent's rule

I/O pin 数量与 gate count 的经验关系：

$$
P=K G^{\beta}
$$

其中：

- $P$：芯片 I/O pins 数量。
- $G$：gate 数量。
- $K$：平均每个 gate 的 I/O 系数。
- $\beta$：Rent exponent，教材给出常见范围约 $0.1\sim0.7$。

### package 基本功能

```text
signal / power I/O + heat removal + mechanical support + environmental protection
```

### 封装电气要求

封装互连应尽量低：

- capacitance
- resistance
- inductance

高速场景还要考虑 characteristic impedance 和 transmission-line behavior。

## 封装互连数量级

### wire bonding

- bonding wire inductance 可达 5 nH 或更高。
- bonding wire inductance 约 1 nH/mm。
- package pin inductance 约 7–40 nH/pin。

### TAB 示例

教材给出的 48 mm TAB circuit 数量级：

$$
L\approx0.3\sim0.5\ \mathrm{nH}
$$

$$
C\approx0.2\sim0.3\ \mathrm{pF}
$$

$$
R\approx50\sim200\ \Omega
$$

### solder bump / MCM

solder bump 数量级：

$$
C\approx0.1\ \mathrm{pF},\qquad L\approx0.01\ \mathrm{nH}
$$

> 需核对：封装寄生强依赖具体 package、bonding style、pin location 和 substrate 材料。

## 封装类型速查

| 类型 | 直觉 |
|---|---|
| DIP | through-hole，成熟但 pin count / 密度受限 |
| PGA | pin grid array，pin count 比 DIP 高 |
| surface mount | 无 through-hole，密度更高但测试/返修更复杂 |
| BGA | solder ball 面阵，适合高 I/O count |
| MCM | 多 die 直接装到 substrate，密度和性能更高但成本高 |

BGA 数量级：

- solder ball pitch 可低至约 $0.8\ \mathrm{mm}$。
- 可支持 multiple 1000's of I/O signals。

## 封装热设计

### thermal resistance

$$
\theta=\frac{\Delta T}{P}
$$

因此：

$$
P\le\frac{\Delta T}{\theta}
$$

典型工作温度范围：

- commercial devices：$0^\circ C\sim70^\circ C$。
- military parts：$-55^\circ C\sim125^\circ C$。

### 40-pin DIP 热阻例子

教材例子：

- natural convection：$\theta\approx38^\circ C/W$。
- forced convection：$\theta\approx25^\circ C/W$。

若允许温差不超过 $75^\circ C$：

- natural convection 下约 $2\ \mathrm{W}$。
- forced convection 下约 $3\ \mathrm{W}$。

### 集成度与热约束

教材给出：

$$
\frac{N_G}{t_p}\le \frac{\Delta T}{\theta E}
$$

其中：

- $N_G$：chip 上 gate 数量。
- $t_p$：propagation delay。
- $\Delta T$：chip 与 environment 最大温差。
- $\theta$：热阻。
- $E$：每个 gate 的 switching energy。

例：当 $\Delta T=100^\circ C$、$\theta=2.5^\circ C/W$、$E=0.1\ \mathrm{pJ}$ 时：

$$
\frac{N_G}{t_p}\le4\times10^5\ \mathrm{gates/ns}
$$

## 高频考点

- 第 2 章重点不是背完整工艺流程，而是理解工艺如何约束版图和电路性能。
- mask 是设计图形与硅片加工之间的接口。
- design rules 是设计者和工艺工程师之间的 contract。
- 光刻是选择性加工的核心方法，会在流程中反复出现。
- ion implantation 比 diffusion 更容易独立控制深度和剂量。
- annealing 用于修复离子注入造成的晶格损伤。
- deposition 加材料，etching 去材料，CMP 平坦化。
- self-aligned gate 使 source/drain 相对 gate 的位置更精确。
- contact/via 会带来面积和寄生代价，避免无谓换层。
- well/substrate contacts 对避免 latchup 和衬底电位漂移很重要。
- DRC clean 只代表几何规则通过，不代表电路功能正确。
- package 是 die 与外部世界的接口，会影响性能、功耗、可靠性、寿命和成本。
- wire bonding 成熟但寄生电感较大；flip-chip / BGA / MCM 可降低互连层级和寄生。
- 封装热阻和功耗共同限制芯片可集成规模与运行频率。
- 高温会加速 leakage、electromigration 和 hot-electron trapping。
