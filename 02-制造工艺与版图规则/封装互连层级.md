# 封装互连层级

## 一句话理解

封装互连层级描述信号从 silicon die 到 package substrate、再到 PCB 的路径。传统封装层级越深、连线越长，寄生电感/电容越大；现代封装通过 TAB、flip-chip、BGA、MCM 等方式减少互连层级和寄生。

## 传统两级封装层级

传统 packaging approach 使用两级互连：

```text
silicon die -> chip carrier / package substrate -> PC board
```

第一级：die-to-package-substrate。  
第二级：package-substrate-to-board。

复杂系统还可能继续连接到 backplane 或 ribbon cable，形成更深的互连层级。

## Level 1：die-to-package-substrate

### Wire bonding

wire bonding 长期是 die 与 package 连接的主流技术。

基本过程：

1. die 背面用导热胶固定到 substrate。
2. chip pads 用 aluminum 或 gold bonding wires 逐个连接到 lead frame。
3. wire-bonding machine 类似“缝纫机”逐条连线。

优点：

- 成熟。
- 自动化程度高。
- 成本相对低。

缺点：

- wires 必须逐个连接，pin count 越高制造时间越长。
- 高 pin count 下避免 wire short 更困难。
- bonding wire 寄生较大且不易精确预测。

教材给出的数量级：

- bonding wire individual inductance 可达 5 nH 或更高。
- bonding wire inductance 约 1 nH/mm。
- package pin inductance 可能约 7–40 nH/pin。

> 需核对：寄生数值依赖具体封装和 pin 位置，笔记只保留教材数量级。

### TAB

Tape Automated Bonding（TAB）使用印刷在 polymer film 上的 metal lead frame。

特点：

- die 连接到 polymer film 上的 printed wiring。
- chip pad 与 film wire 通过 solder bumps 连接。
- 所有连接可同时完成，自动化程度高。

优点：

- lead pitch 更小。
- 支持更高 lead count。
- 消除长 bonding wires，电气性能更好。

教材给出的一个 48 mm TAB circuit 数量级：

- $L\approx0.3\sim0.5\ \mathrm{nH}$
- $C\approx0.2\sim0.3\ \mathrm{pF}$
- $R\approx50\sim200\ \Omega$

> 需核对：该数值来自课本示例，具体 TAB 结构可能差异很大。

### Flip-chip

flip-chip mounting 把 die 翻转，直接用 solder bumps 连接到 substrate。

优点：

- I/O pad 不必只放在 die 边缘，可分布在芯片表面。
- 缩短 die-to-substrate 互连长度。
- 电气性能优于 wire bonding。
- 更适合处理 power distribution 和 clock distribution。

直觉：

```text
wire bond = 边缘引线
flip-chip = 面阵焊点
```

## Level 2：package-substrate-to-board

### Through-hole mounting

through-hole mounting 把 package pins 插入 PCB 孔中，再焊接固定。

典型代表：

- DIP（dual-in-line package）
- PGA（pin-grid-array package）

优点：

- 机械连接强。
- 可靠、成熟。

缺点：

- packing density 差。
- through-hole 会阻挡 PCB 下方走线。
- pin pitch 受机械约束，典型 through-hole pitch 约 2.54 mm。
- 大 pin count 下需要更多 routing layers。

### Surface-mount technique

surface-mount 不需要穿孔，package 直接贴装在 board 表面。

优点：

- 取消 through-holes，释放布线空间。
- lead pitch 可降低。
- 可以在 board 两面装配。
- 改善 board 机械强度。

缺点：

- chip-board connection 机械强度较弱。
- 装配和返修设备更复杂。
- package pins 背面不可直接访问，测试 probing 更困难。

常见 surface-mount package：

- small-outline IC。
- PLCC。
- leadless chip carrier。

### BGA

当 I/O 数量超过几百时，solder balls 通常比 pins 更适合作为 package-to-board 互连。

BGA（ball-grid array）特点：

- solder balls 连接 package 与 board。
- 也可用 solder bumps 连接 die 与 package substrate。
- area array interconnect 让 I/O density 不再只受封装边缘长度限制。
- solder ball pitch 可低到约 0.8 mm。
- 支持 multiple 1000's of I/O signals。

## MCM：die-to-board / die-to-substrate

MCM（multichip module）试图减少封装层级，把多个 die 直接装在 wiring substrate 上。

动机：

```text
减少封装互连层级 -> 减少寄生 -> 提高性能和密度
```

优点：

- packing density 更高。
- 性能更好。
- 减少 chip-carrier layer 的寄生。
- 缩短 global wiring length。
- 降低输出 pad 负载，可减少大输出驱动器和功耗。

缺点：

- 制造复杂。
- 成本高。
- 只有在 extreme performance 或 dense housing 需求下才划算。

教材给出的 solder bump 数量级：

- capacitance 约 0.1 pF。
- inductance 约 0.01 nH。

## 设计直觉

- 互连层级越深、连线越长，封装寄生越严重。
- wire bonding 成熟但寄生电感大。
- TAB 和 flip-chip 通过减少长引线改善电气性能。
- BGA 用面积阵列解决高 pin count 问题。
- MCM 用更高集成封装密度换取性能，但成本高。

## 易错点

- 不要把 package-to-board 和 die-to-package 混成同一级问题。
- 不要只看封装 pin 数，还要看 power/ground pins 与信号 pins 的比例。
- 不要忽略 bonding wire inductance，它会在高速 I/O 和 simultaneous switching 中放大 supply/ground noise。
- 不要认为 MCM 总是最佳；它的经济性和制造复杂度是主要限制。
