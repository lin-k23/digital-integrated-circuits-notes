# 工艺趋势总览

## 一句话理解

工艺趋势（process technology trends）不是完全抛弃 CMOS，而是在原有 CMOS 流程上持续加入局部改进：更低电阻互连、更低介电常数绝缘层、更低寄生的器件结构，以及更高维度的集成方式，用来缓解互连、功耗、热和 I/O 的限制。

## 来源范围

- 来源 PDF：`source/ch2_en.pdf`
- 对应小节：2.5 Perspective — Trends in Process Technology

## 教材主线

教材先说明，现代 CMOS 流程大体仍沿用前面章节描述的制造框架，但会加入一些局部改进，例如：

- 工艺步骤顺序可能调整。
- 可使用 single-well 或 dual-well 变体。
- 隔离方式可使用 grown field oxide 或 trench approach。
- 可能引入 LDD（Lightly Doped Drain）。
- 可在 polysilicon interconnect、source/drain 区上覆盖 silicide，如 $TiSi_2$，以降低电阻。

这些改进说明：

```text
短期内 CMOS 主流程仍然延续，但关键瓶颈由材料、互连、器件结构和三维集成逐步缓解。
```

## 短期趋势

教材短期趋势主要包括：

- [[铜互连与低k介质]]
- [[SOI工艺]]

它们都服务于同一个目标：

```text
降低寄生 R/C，提升速度，降低功耗，延长 CMOS 缩放寿命
```

## 长期趋势

更长期的趋势包括：

- 新器件：organic transistors、molecular switches、quantum devices 等。
- [[三维集成与2.5D集成]]。
- chip / substrate / package / board 边界逐渐模糊。

教材没有预测哪种新器件一定胜出，而是强调三维集成是值得关注的发展方向。

## 设计直觉

- 工艺趋势不是孤立技术名词，而是为了解决设计瓶颈。
- copper 主要降低互连电阻。
- low-k 主要降低互连电容。
- SOI 主要降低器件寄生并改善开关特性。
- 3D / 2.5D integration 主要减少存储、计算、I/O 之间的距离。
- 这些技术都带来制造、成本、热、良率或可靠性代价。

## 易错点

- 不要把工艺趋势理解为“CMOS 立刻被替代”；教材强调短期内仍是 CMOS 主流程的局部改进。
- 不要只看材料优点；每种材料/结构都有制造和可靠性挑战。
- 不要把 3D integration 和封装混为一谈；但二者边界确实越来越模糊。
- 不要把工艺改进与电路设计割裂，低 R/C 互连、SOI 和 3D 都直接影响时序、功耗和热设计。
