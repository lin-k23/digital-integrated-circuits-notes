# 公式与考点速查

> 本文件用于汇总第 7 章时序逻辑的关键公式、定义和易错点。当前已整理 7.1 与 7.2.1–7.2.2 的核心概念：时序参数、存储单元分类、双稳态、亚稳态和 SR 锁存器。

## 组合逻辑与时序逻辑

组合逻辑：

$$
Y=f(X_1,X_2,\ldots,X_N)
$$

时序逻辑：

$$
Y=f(X_1,X_2,\ldots,X_N,\text{State})
$$

下一状态：

$$
\text{Next State}=g(\text{Current State},\text{Inputs})
$$

## 寄存器核心时序参数

- $t_{su}$：setup time，时钟边沿前数据必须稳定的时间。
- $t_{hold}$：hold time，时钟边沿后数据必须继续稳定的时间。
- $t_{c-q}$：clock-to-Q propagation delay，时钟边沿到输出 $Q$ 稳定的延迟。

## 最小时钟周期

若组合逻辑最坏传播延迟为 $t_{p,logic}$，同步路径满足：

$$
T \ge t_{c-q}+t_{p,logic}+t_{su}
$$

因此：

$$
f_{max}\le \frac{1}{T_{min}}
$$

## 保持时间约束

若寄存器最小 clock-to-Q 延迟为 $t_{cd,register}$，组合逻辑最小延迟为 $t_{cd,logic}$，则：

$$
t_{cd,register}+t_{cd,logic}\ge t_{hold}
$$

setup 和 hold 的核心区别：

```text
setup: 最长路径太慢
hold: 最短路径太快
```

## 存储单元分类

### foreground / background memory

- foreground memory：嵌入逻辑中的寄存器、锁存器、寄存器堆单元。
- background memory：大规模集中式存储阵列，如 SRAM cache。

### static / dynamic memory

- static memory：靠正反馈保持状态，只要供电存在即可保持。
- dynamic memory：靠寄生电容暂存电荷，状态保持时间有限。

### latch / register

| 类型 | 特征 |
|---|---|
| latch | 电平敏感，透明窗口内 $Q$ 跟随 $D$ |
| register / flip-flop | 边沿触发，只在时钟边沿采样 |

## 双稳态与亚稳态

交叉耦合反相器形成双稳态：

```text
两个稳定点 → 存储 0 / 1
一个亚稳态点 → 不稳定平衡
```

稳定点附近：

$$
|G_{loop}|<1
$$

亚稳态点附近：

$$
|G_{loop}|>1
$$

直觉：

- 稳定点：扰动被衰减。
- 亚稳态点：扰动被放大。

## SR 锁存器

### NOR-based SR

| S | R | Q(next) | 说明 |
|---|---|---|---|
| 0 | 0 | 保持 | hold |
| 1 | 0 | 1 | set |
| 0 | 1 | 0 | reset |
| 1 | 1 | 非法 | $Q$ 与 $\bar{Q}$ 同时被强制为 0 |

### NAND-based SR

NAND-based SR 通常低有效：

- $\bar{S}=0$：set。
- $\bar{R}=0$：reset。
- $\bar{S}=\bar{R}=0$：forbidden state。

## 高频考点

- 时序逻辑的输出依赖当前输入和过去状态。
- 寄存器把组合逻辑切成同步时钟周期。
- 最小时钟周期由 $t_{c-q}+t_{p,logic}+t_{su}$ 决定。
- hold violation 不是降频能解决的问题。
- latch 是电平敏感，register / flip-flop 通常是边沿触发。
- static memory 靠正反馈，dynamic memory 靠电荷暂存。
- 双稳态电路有两个稳定状态和一个亚稳态点。
- 亚稳态不能完全消除，只能降低概率或增加解析时间。
- SR latch 的 forbidden state 必须避免。
- clocked SR latch 仍是 latch，不是 edge-triggered register。
