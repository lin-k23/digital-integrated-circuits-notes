# 07-时序逻辑

本章对应教材第 7 章“时序逻辑电路设计（Designing Sequential Logic Circuits）”，用于整理锁存器、触发器、寄存器、时序约束、流水线和时钟策略。

## 本章目标

- 区分组合逻辑与时序逻辑，理解为什么系统需要保存状态。
- 掌握同步时序系统中寄存器、组合逻辑和全局时钟的基本结构。
- 掌握建立时间、保持时间、clock-to-Q 延迟、污染延迟和最小时钟周期约束。
- 区分 latch 与 register、static 与 dynamic memory、foreground 与 background memory。
- 理解双稳态、正反馈和亚稳态的基本原理。
- 掌握 SR latch / SR flip-flop 的基本行为与 forbidden state。
- 为后续动态寄存器、流水线、Schmitt trigger 和时钟策略整理打基础。
