# 双稳态原理与亚稳态

## 一句话理解

双稳态电路（bistable circuit）依靠正反馈形成两个稳定工作点，分别表示 0 和 1；两个稳定点之间还有一个不稳定的亚稳态点，任何微小扰动都会被正反馈放大，使电路最终落入某个稳定状态。

## 双稳态的基本结构

最基本的静态存储结构是两个反相器交叉耦合：

```text
inverter1 output → inverter2 input
inverter2 output → inverter1 input
```

这个结构有两个稳定状态：

- 一个节点为 0，另一个节点为 1。
- 一个节点为 1，另一个节点为 0。

因此可以保存 1 bit 信息。

## 正反馈

正反馈（positive feedback）是双稳态的核心。

如果电路稍微偏向某个稳定状态，反馈会进一步放大这个偏移，使其更快接近稳定点。

直觉：

```text
微小偏差 → 被第一个反相器放大 → 被第二个反相器再次放大 → 偏差继续变大
```

## 三个工作点

两个交叉耦合反相器的 combined VTC 会出现三个交点：

- A：稳定点。
- B：稳定点。
- C：亚稳态点。

A 和 B 对应两个可保存的逻辑状态。C 位于翻转区附近，是不稳定点。

## 亚稳态

亚稳态（metastability）是指电路位于不稳定平衡点附近，短时间内无法确定最终落到 0 还是 1。

在交叉耦合反相器中，若电路恰好处于 C 点：

- 理想对称情况下可能暂时停留。
- 真实电路中任何噪声、失配或热扰动都会使其偏离 C 点。
- 偏离后正反馈会把它推向 A 或 B。

## 稳定条件直觉

教材给出的核心直觉是：

- 在稳定点 A / B 附近，loop gain 小于 1，小扰动会被衰减。
- 在亚稳态点 C 附近，loop gain 大于 1，小扰动会被放大。

因此：

```text
稳定点：扰动消失
亚稳态点：扰动被放大
```

## 如何改变存储状态

双稳态电路本身会保持当前状态。若要改变状态，必须施加外部触发，使原来的稳定点暂时失稳。

常见方法：

- 给某一内部节点施加触发脉冲。
- 用 set / reset 输入强制某个输出。
- 用 clocked access transistor 在透明窗口内写入新值。

触发脉冲只需要持续足够长，使正反馈开始向目标状态再生即可。

## 设计直觉

- 存储的本质是正反馈带来的两个稳定状态。
- 亚稳态无法完全消除，只能降低发生概率或给足解析时间。
- 触发器和同步器设计中，亚稳态是跨时钟域和异步输入的核心风险。
- 静态存储只要电源存在就能保持状态，但写入时必须能“打败”反馈。

## 易错点

- 亚稳态不是逻辑 0 或 1，而是暂时处于不确定中间状态。
- 亚稳态最终通常会解析到 0 或 1，但解析时间不可严格上界。
- 双稳态电路保存状态依赖电源；断电后状态丢失。
- 强反馈有利于保持状态，但也会增加写入难度。
